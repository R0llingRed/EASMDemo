第 0 轮（第 1-2 周）基础工程化与环境

  - 模块范围：项目骨架、CI、基础配置、部署脚手架
  - 关键任务：代码结构、FastAPI 基础服务、数据库连接、任务队列骨架、Docker Compose
  - 交付物：/server 服务框架、/worker 任务执行框架、docker-compose.yml
  - 验收标准：服务可启动、健康检查可用、DB 可连接、队列可投递

  第 1 轮（第 3-4 周）资产治理基础（最小可用）

  - 模块范围：项目/资产导入、资产实体模型（domain/ip/url）
  - 关键任务：项目管理 API、资产录入 API、基础去重逻辑、资产列表 UI
  - 交付物：资产录入与列表功能、核心表结构
  - 验收标准：批量导入可用、重复资产不重复入库、列表分页可用

  第 2 轮（第 5-6 周）扫描执行最小链路

  - 模块范围：子域名收集 + DNS 解析 + 端口扫描（MVP）
  - 关键任务：任务生成器、队列执行器、结果归一化
  - 交付物：基础扫描结果入库、任务状态页
  - 验收标准：可从输入域名生成子域名、解析 IP、扫描端口

  第 3 轮（第 7-8 周）Web 探测与指纹

  - 模块范围：http 探测、指纹识别、截图存档
  - 关键任务：httpx 接入、指纹库整合、截图存储
  - 交付物：Web 资产表、指纹结果、截图链接
  - 验收标准：可在 Web 资产列表看到状态码/指纹/截图

  第 4 轮（第 9-10 周）漏洞发现基础

  - 模块范围：Nuclei 扫描、漏洞表、漏洞列表
  - 关键任务：Nuclei 任务、结果解析、严重度映射
  - 交付物：漏洞记录管理、漏洞列表页
  - 验收标准：Nuclei 扫描结果可追踪、可过滤

  第 5 轮（第 11-12 周）数据融合与可信度

  - 模块范围：资产关系、观测记录、可信度评分
  - 关键任务：Observation 模型、可信度计算、误报标记
  - 交付物：结果质量字段、误报处理流程
  - 验收标准：可标记误报、可信度可计算并展示

  第 6 轮（第 13-14 周）任务编排升级

  - 模块范围：DAG 编排、事件驱动、策略中心
  - 关键任务：DAG 模板、事件触发机制、优先级调度
  - 交付物：策略中心 UI、DAG 可视化
  - 验收标准：事件触发后任务可按 DAG 自动串联

  第 7 轮（第 15-16 周）风险评估与告警

  - 模块范围：风险评分、告警策略、多渠道通知
  - 验收标准：严重风险可触发通知
